나만의 10초 영상 인트로 만들기

프로젝트 개요

이 가이드는 애프터 이펙트의 핵심 기능인 모션 그래픽, 3D, VFX, AI를 종합적으로 활용하여 가상의 스마트폰 제품을 소개하는 10초 길이의 세련된 영상 인트로를 제작하는 과정을 안내합니다. 각 단계를 따라가며 실전 감각을 익혀보세요.

📋 준비물

  • ✓
    제품 이미지 (.png)

    배경이 투명한 정면 스마트폰 이미지. (예: Freepik, Pngtree)

  • ✓
    화면 속 영상

    화면에 합성할 5-10초 길이의 영상 클립. (예: Pexels, Pixabay)

  • ✓
    배경음악 (BGM)

    영상 분위기에 맞는 세련된 배경음악.

1단계: 배경 모션 그래픽 만들기 (Shape Layer)

영상의 기초가 되는 배경을 만듭니다. 셰이프 레이어와 '선 다듬기' 기능을 활용하여 동적인 라인 애니메이션을 제작해 깊이감을 더합니다.

1. 컴포지션 생성

`1920x1080` 해상도, 길이 `10초`의 새 컴포지션을 만듭니다.

2. 배경 생성

`레이어 > 새로 만들기 > 단색(Solid)`을 선택하여 어두운 색상의 배경 레이어를 추가합니다.

3. 라인 애니메이션

새로운 셰이프 레이어를 만들고 펜 도구(G)로 여러 개의 선을 그립니다. 속성에서 선 다듬기(Trim Paths)를 추가하여 선들이 동적으로 나타났다 사라지는 애니메이션을 만듭니다.

2단계: 3D 공간에 제품 배치 및 애니메이션

주인공인 스마트폰을 3D 공간에 배치하고 카메라를 이용해 입체적인 움직임을 만듭니다. 이를 통해 시청자의 시선을 자연스럽게 제품으로 유도합니다.

1. 제품 배치 및 3D 변환

스마트폰 이미지를 타임라인에 올리고 3D 레이어 스위치(정육면체 아이콘)를 켭니다.

2. 카메라 생성

`레이어 > 새로 만들기 > 카메라(Camera)`를 추가하여 3D 공간을 제어할 준비를 합니다.

3. 제품 애니메이션

스마트폰 레이어의 `변형(Transform)` 속성에서 키프레임을 조절합니다. 0초에는 멀리 있다가 3초에 화면 중앙으로 회전하며 다가오도록 `위치(Position)`와 `Y 회전` 값을 설정합니다.

3단계: 화면 합성하기 (VFX - Motion Tracking)

움직이는 스마트폰 화면에 다른 영상을 자연스럽게 합성합니다. 모션 트래킹의 '원근 코너 고정' 기능을 사용하여 마치 원래부터 하나의 영상이었던 것처럼 만듭니다.

1. 화면 영상 준비

합성할 영상을 별도의 컴포지션('Screen_Comp')으로 미리 만듭니다.

2. 모션 트래킹

메인 컴포지션의 스마트폰 레이어를 선택하고 `창 > 추적기` 패널에서 `동작 추적`을 실행합니다.

3. 코너 핀 적용 및 분석

추적 유형을 원근 코너 고정(Perspective Corner Pin)으로 설정하고, 4개의 추적점을 스마트폰 화면 모서리에 맞춘 후 움직임을 분석합니다.

4. 데이터 적용

분석 완료 후, `대상 편집`에서 'Screen_Comp'를 선택하고 `적용`을 누릅니다.

4단계: 타이틀 애니메이션 추가

제품의 이름과 슬로건에 생동감을 불어넣습니다. 텍스트 애니메이션 기능을 활용하여 메시지를 효과적으로 전달하는 동적인 타이틀을 제작합니다.

1. 텍스트 생성

제품 이름(예: "GALAXY AI")과 슬로건(예: "새로운 시대를 열다")을 각각 별도의 텍스트 레이어로 만듭니다.

2. 애니메이션 적용

제품 이름에는 '타이핑 효과' 프리셋을 적용하고, 슬로건은 `애니메이트 > 위치`를 추가하여 아래에서 위로 부드럽게 떠오르는 애니메이션을 만듭니다.

5단계 (보너스): AI 기능으로 완성도 높이기

애프터 이펙트의 강력한 AI 기능을 활용하여 영상의 완성도를 한 차원 높여봅니다. 단순 반복 작업을 자동화하고, 복잡한 합성을 손쉽게 처리할 수 있습니다.

1. 배경 정리

배경 영상에 불필요한 요소가 있다면, 내용 인식 채우기(Content-Aware Fill)를 사용하여 감쪽같이 제거합니다.

2. 인물 합성

로토 브러시 2를 사용하여 스마트폰을 터치하는 손 영상의 배경을 분리하고, 제품과 상호작용하는 것처럼 자연스럽게 합성하여 깊이를 더합니다.

최종 마무리 및 렌더링

모든 시각 요소가 완성되었습니다. 이제 사운드를 추가하고 전체적인 타이밍을 조절하여 프로젝트를 최종적으로 완성할 차례입니다.

1. 배경음악 추가

준비한 배경음악(BGM) 파일을 프로젝트에 가져와 타임라인에 배치합니다.

2. 타이밍 조절

음악의 비트와 영상의 애니메이션이 잘 맞도록 각 레이어의 키프레임 타이밍을 미세하게 조절합니다.

3. 렌더링

모든 작업이 끝나면 `파일 > 내보내기 > Adobe Media Encoder 대기열에 추가`를 통해 최종 영상 파일로 출력합니다.
