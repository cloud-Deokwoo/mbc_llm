미래형 HUD 인터페이스 만들기

이 인터랙티브 가이드는 실제 영상 위에 미래적인 홀로그램 인터페이스(HUD)를 합성하는 VFX 제작 과정을 안내합니다. 각 단계를 클릭하여 상세 내용을 확인하고, 프로젝트를 완성해보세요.

프로젝트 개요

실제 영상의 움직임을 분석하고, 그 위에 디자인된 그래픽을 3D 공간에 자연스럽게 합성하는 실무적인 VFX 제작 과정을 다룹니다. 모션 트래킹, 셰이프 애니메이션, 3D 레이어, 그리고 AI 로토 브러시 기능을 유기적으로 결합하는 능력을 기릅니다.

준비물

  • 📹 배경 영상: 추적할 움직임이 있는 인물 또는 도시 영상. (추천: Pexels, Pixabay)
  • 🔊 사운드 (선택): HUD 작동 효과음. (추천: Mixkit)

1단계: 영상 분석 및 움직임 추적 (VFX)

영상의 특정 지점의 움직임을 추적하여 데이터를 추출합니다. 이 데이터는 나중에 그래픽을 연동시키는 기준점이 됩니다.

  1. 컴포지션 생성: 준비한 배경 영상을 1920x1080, 길이 15초의 새 컴포지션에 배치합니다.
  2. Null 개체 생성: 레이어 > 새로 만들기 > Null 개체를 생성합니다. 이 보이지 않는 레이어는 추적 데이터를 저장하는 '닻' 역할을 합니다.
  3. 동작 추적 실행:
    • 배경 영상 레이어를 선택하고 창 > 추적기 패널에서 동작 추적을 실행합니다.
    • '트랙 포인트'를 추적할 지점(예: 손등, 건물 모서리)에 정확히 위치시킵니다.
    • 앞으로 분석 버튼을 눌러 영상의 움직임을 분석합니다.
  4. 데이터 적용: 분석 완료 후, 대상 편집을 눌러 생성한 Null 개체를 선택하고 적용 버튼을 누릅니다. 이제 Null 개체가 영상의 움직임을 그대로 따라갑니다.

2단계: HUD 그래픽 요소 제작 (Motion Graphics)

미래적인 느낌의 HUD 그래픽 요소들을 별도의 컴포지션에서 디자인하고 애니메이션을 적용합니다.

  1. 새 컴포지션: HUD_Elements_Comp 라는 이름의 새 컴포지션을 만듭니다.
  2. 그래픽 디자인:
    • 셰이프 레이어: 원, 사각형, 선 등을 디자인합니다. 칠(Fill)은 '없음', 선(Stroke)은 밝은 청록색이나 흰색으로 설정합니다.
    • 애니메이션: 선 다듬기(Trim Paths)를 추가하여 선이 그려지는 애니메이션을 만들고, 텍스트 레이어에 타이핑 효과를 적용합니다.
  3. 효과 추가: 완성된 그래픽 레이어 전체에 효과 및 사전 설정 패널에서 광선(Glow) 효과를 적용하여 빛나는 느낌을 더합니다.

3단계: 3D 공간에 합성 및 연동 (3D & Compositing)

제작한 HUD 그래픽을 메인 영상에 가져와 3D 공간에 배치하고, 추적된 움직임과 연동시킵니다.

  1. 컴포지션 통합: 1단계에서 만든 메인 컴포지션으로 돌아와 HUD_Elements_Comp를 타임라인에 배치합니다.
  2. 3D 레이어 전환: HUD_Elements_Comp 레이어의 3D 레이어 스위치를 켭니다. 이제 2D 그래픽이 3D 공간에 배치됩니다.
  3. 움직임 연동 (Parenting): HUD_Elements_Comp 레이어의 '돼지꼬리' 아이콘(Parent Pick Whip)을 1단계에서 만든 Null 개체에 연결합니다. 이제 HUD가 Null 개체의 움직임을 완벽하게 따라다닙니다.
  4. 공간 배치 및 애니메이션:
    • 위치(Position), 회전(Rotation), 비율(Scale) 값을 조절하여 영상 속 공간에 자연스럽게 배치합니다.
    • 불투명도(Opacity)에 키프레임을 추가하여 HUD가 서서히 나타나는 애니메이션을 만듭니다.

4단계: AI를 이용한 깊이감 표현 (AI - Roto Brush)

AI 기반의 로토 브러시 도구를 사용하여 영상 속 인물이나 사물이 HUD 그래픽 앞을 가리도록 만들어, 합성에 깊이와 현실감을 더합니다.

  1. 영상 복제: 메인 컴포지션에서 배경 영상 레이어를 복제(Ctrl+D)하여 HUD_Elements_Comp 레이어 위로 올립니다.
  2. 인물 분리 (로토스코핑):
    • 맨 위에 있는 배경 영상 클립을 더블 클릭하여 레이어 패널에서 엽니다.
    • 로토 브러시 도구를 사용하여 HUD 그래픽을 가려야 할 부분(예: 인물의 손가락)만 선택하여 배경과 분리합니다.
  3. 결과 확인: 다시 메인 컴포지션으로 돌아와 재생합니다. 이제 HUD 그래픽의 일부가 인물 뒤에 있는 것처럼 보여 훨씬 입체적인 합성이 완성됩니다.
